## 1、彩票是什么，能预测么
-20多年前和微软的一个团队合作的时候他们说中国彩票的算法是他们做的（忘记是哪个彩种了）没有猫腻。我信的，打彩票和出中奖号应该是没有问题的，有问题的是其他环节

-不考虑阴谋论的话，大部分的彩票就是碰运气（足彩不算），所以他是理财的一小部分，不能拿他赌明天

-能预测吗？能知道下一期很大可能不会出现什么（极特殊形态），但这只占据所有可能的极少一部分，所以价值也不大。至于能预测下一期号码是什么，这就是个玩儿

## 2、这个项目和其他同类有什么不一样
-github上也看到很多类似的，甚至用LLM做预测的，但事实上都是研究技术本身而不是真的用他来发财。这个逻辑是对的

-对于特殊形态的分析很多人都做过，但我发现“特殊形态的出现情况”本身也是一种结构维度，他也有自己的数学分布，代码里对他做了加权。我和其他同类代码最大的区别应该就是这个。其他没差太多

## 3、怎么运行
-先安装依赖
pip install -r requirements.txt

-然后增量更新数据
python src\ssq_crawler.py inc

-再打开src\ssq.html，点击“同步数据”，选取那个csv
